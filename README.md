# 📈 InvestIQ PROTOTYPE

AI-powered Investment Decision Support System for NSE stocks using Machine Learning and Explainable AI.

## Features

- Live NSE stock analysis
- Technical indicators (RSI, MACD, SMA, Volatility)
- XGBoost stock direction prediction
- SHAP Explainable AI
- BUY / HOLD / SELL recommendation engine
- Risk analysis metrics
- Interactive Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- XGBoost
- SHAP
- pandas
- yfinance
- matplotlib

## Project Structure

```bash
.
├── app.py
├── data_fetch.py
├── indicators.py
├── model.py
├── explain.py
├── recommendation.py
├── requirements.txt
└── README.md
```

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
